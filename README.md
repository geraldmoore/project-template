# Project Template

A [Copier](https://copier.readthedocs.io/) template for Python projects using [uv](https://docs.astral.sh/uv/).

## Prerequisites

- [copier](https://copier.readthedocs.io/) (`pip install copier`)

## Usage

### Generate a new project

```bash
copier copy gh:geraldmoore/project-template /path/to/your/project
```

### Update an existing project

In the project repository:

```bash
copier update
```
